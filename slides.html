Slide 1

SocialCare

This is our SocialCare website project. We created it to show the main services offered by a social support organization in a simple and clear way.

Purpose

To present social care services in an easy way for users to understand.

Users

People looking for support, information, and contact details.

Tools

Built using HTML and Tailwind CSS.

Slide 2

Pages In The Website

Home page: welcomes the user and gives a short introduction.

About page: explains what SocialCare is about.

Services page: shows the main services provided.

Contact page: allows users to send a message.

Main Idea

The website is meant to be simple, direct, and easy to use.

Slide 3

Main Services

Community Help

Help for families and individuals in need.

Education

Support through learning resources and education programs.

Health Support

Guidance and support related to healthcare services.

Counseling

Support for emotional and mental wellbeing.

Food Programs

Food assistance for families and community members.

Slide 4

Tools Used

  • HTML5 to create the pages and structure
  • Tailwind CSS to style the website
  • Responsive layout so it can work on different screen sizes

Why We Chose Them

They are simple to use and good for a static website project.

These tools helped us build the project faster and keep the design neat and responsive.
